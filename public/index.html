FlixHQ API Endpoints

1. Home
/home
Example: /home
2. Search
/search?q={searchQuery}&page={pageIndex}
Example: /search?q=avengers&page=1
3. Movie Details
/movie/{movieId}
Example: /movie/19782
4. Seasons
/movie/{movieId}/seasons
Example: /movie/90553/seasons
5. Episodes
/movie/{movieId}/episodes?seasonId={seasonId}
Example: /movie/90553/seasons/episodes?seasonId=season-1
6. Servers
/movie/{movieId}/servers?episodeId={episodeId}
Example: /movie/90553/episodes?seasonId=72838
6. Servers
/movie/{movieId}/servers?episodeId={episodeId}
Example: /movie/90553/servers?episodeId=1310353
7. Sources
/movie/{movieId}/sources?serverId={serverId}
Example: /movie/90553/sources?serverId=9179590
